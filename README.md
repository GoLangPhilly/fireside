fireside
========

fireside is an XMPP server built in go.

It will consist of the following parts:

* a core xmpp server
* a plugin infrastructure based on XEP-00##
* a persistence layer (leveldb ?)
* interfaces with other messaging servers:
  * IRC
  * WebSockets
  * other?
